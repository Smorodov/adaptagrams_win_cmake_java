\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_avoid_1_1_polygon}{Polygon}} which just references its points from other Polygons. 

This type of \mbox{\hyperlink{class_avoid_1_1_polygon}{Polygon}} is used to accurately represent cluster boundaries made up from the corner points of shapes.
